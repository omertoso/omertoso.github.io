# MDT是什么

MDT 是自动执行桌面和服务器部署的工具、流程和指南的统一集合。 你可以使用它创建引用映像，或将其用作完整的部署解决方案。 MDT 现在是 IT 专业人员可用的最重要的工具之一。

- 安装wds服务，.NET Framework 3.5组件
- 配置MDT服务器
- 下载安装MDT、ADK组件
- 导入ISO、驱动库
- 配置任务队列
