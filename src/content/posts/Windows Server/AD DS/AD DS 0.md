# What is AD ?

Active Directory（AD）是由微软开发的目录服务，用于Windows域网络。它最初在Windows 2000 Server中引入，现在是Windows Server操作系统的核心功能之一。Active Directory用于**管理和组织网络中的用户、计算机和其他资源，提供身份验证、授权和目录服务**。
